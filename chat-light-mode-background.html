CelesteOS Chat Light Mode Background

This is the light mode background used in the CelesteOS chat interface when users are in conversation mode. The background consists of two layered elements with subtle animations.

Background Specifications:

Base Layer: #fcfeff

Gradient: linear-gradient(180deg, rgba(252, 254, 255, 0.9) 0%, rgba(255, 255, 255, 1) 100%)

Animation: fadeToWhite 1.2s cubic-bezier(0.22, 0.61, 0.36, 1)

Transition: all 1200ms ease-out
